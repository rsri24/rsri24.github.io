Featured Work

§ 03

The Data Product Factory

AGENTIC AI · BUILT SOLO

An agentic workflow that automates the journey from raw source-system schemas to business-ready data products. It generates metadata catalogs, semantic knowledge graphs, and domain models, then produces ODP / FDP designs and deployable PostgreSQL and Neo4j artifacts — collapsing what used to be a manual, multi-week data-modeling exercise into an orchestrated agent pipeline. Built end to end using Slingshot for agent orchestration and Bodhi Studio for the workflow build, this is the kind of AI-native data modernization tooling I want to bring into a larger platform team.
